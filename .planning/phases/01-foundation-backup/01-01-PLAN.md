## Objective

Create timestamped backups of both source projects, copy autow-booking as the base codebase, and initialize the business-dashboard project structure with clean git history.

## Execution Context

**Phase**: 1 of 8 (Foundation & Backup)
**Research Required**: No (internal work, copying existing code)
**Estimated Scope**: Small (5 tasks, ~30 minutes execution)

**Source Projects**:
- `D:\Projects-AI\autow-booking` - Full booking/invoicing system (Next.js 14)
- `D:\Projects-AI\AUTOW-ONLINE\autow-parts-bot` - Landing page + signup flow

**Backup Location**: `D:\Projects-AI\backups\business-dashboard-init\`

## Context

This is the foundational step for the Business Dashboard SaaS. We must preserve the working autow-booking system before adapting it for multi-tenancy. The autow-parts-bot landing/signup system will be referenced later (Phase 8) but backed up now for safety.

**Critical**: The "adapt, don't rebuild" approach means we copy the working codebase and modify it, rather than starting fresh.

### Task 2: Backup autow-booking (Source Codebase)

Create timestamped zip archive of the complete autow-booking project.

```bash
cd "/d/Projects-AI"
TIMESTAMP=$(date +%Y%m%d_%H%M%S)
# Exclude node_modules, .next, .git for smaller archive
zip -r "backups/business-dashboard-init/autow-booking_${TIMESTAMP}.zip" autow-booking \
  -x "autow-booking/node_modules/*" \
  -x "autow-booking/.next/*" \
  -x "autow-booking/.git/*"
```

**Verification**:
- Archive exists with timestamp in filename
- Archive contains app/, components/, database/ directories

### Task 4: Copy autow-booking to business-dashboard

Copy the autow-booking codebase into business-dashboard, excluding git history, node_modules, and build artifacts.

```bash
cd "/d/Projects-AI"
# Copy source files (excluding .git, node_modules, .next, .vercel)
rsync -av --progress autow-booking/ business-dashboard/ \
  --exclude='.git' \
  --exclude='node_modules' \
  --exclude='.next' \
  --exclude='.vercel' \
  --exclude='.env.local' \
  --exclude='.env.vercel'
```

**Verification**:
- `business-dashboard/app/` exists
- `business-dashboard/components/` exists
- `business-dashboard/package.json` exists
- No `node_modules/` or `.next/` copied
